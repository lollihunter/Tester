Пояснительная записка

Тестер Python-скриптов

Данная программа - тестировщик программ на языке Python 3.x с пользовательским интерфейсом.
Поддерживается юнит-тестирование (проверка отдельных функций на работоспособность);
классическое тестирование (проверка правильности вывода программы при конкретном наборе входных данных);
удобные операции с тестами и наглядный лог работы.

Функции программы:
1. Окна
 1.1 Test log - окно, в котором показывается лог одного конкретного тестирования. Выводится время прохождения
 каждого теста и различия между пользовательским и эталонным выводом в случае ошибки
 1.2 Test results - вердикт тестирования
 1.3 Окно в верхнем левом углу показывает тесты, лежащие в данной директории
 1.4 Окно в нижнем левом углу - окно функций конкретного python-скрипта
2. Опции
 2.1 Экспорт лог-файла - файл, куда будет записан результат из лог-окна
 2.2 Вывод в файл - опция, применяемая в том случае, если скрипт принимает на вход не stdin, а другой файл
 (аналогично с stdout)
 2.3 TL - ограничивает время выполнения одного теста (в случае превышения тест считается непройденным)
 2.4 Прекращать работу в случае ошибки - закончить тестирование досрочно, если один из тестов не будет пройден
3. Кнопки
 3.1 Запустить - начать проверку скрипта
 3.2 Показать тесты - вывести в таблицу (1.3) тесты в данной директории. Тест имеет формат x.in/x.out.
 3.3 Удалить тест - удалить выделенный тест из директории (необходимо выделить строку с желаемым тестом)
 3.4 Добавить тест - добавить тест формата x.in/x.out (используя данные в окнах "входные данные" и "выходные данные")
 3.5 Изменить тест - изменить тест формата x.in/x.out (используя данные в окнах "входные данные" и "выходные данные"),
 необходимо выделить строку с желаемым тестом
 3.6 Разделитель - изменить знак разделения аргументов во входных данных, если выбрана опция юнит-теста
 3.7 Загрузить список функций - показать функции скрипта в окне (1.4)
